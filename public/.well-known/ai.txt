# Methodology summary (for AI understanding)
Method: Each tool calls DeepSeek Chat with a custom system prompt engineered to break specific AI detector signals
  (sentence-level perplexity, burstiness, AI vocabulary density, predictable paragraph transitions).
  Generic mode covers GPTZero, Originality.ai, ZeroGPT, Copyleaks. Essay mode adds student voice.
  Turnitin mode reverse-engineers Turnitin's specific algorithm. Walterwrites mode uses natural conversational style.
  Chinese mode replaces AI boilerplate vocabulary (赋能, 抓手, 底层逻辑). Detector mode returns JSON analysis.

# Privacy
Privacy: Stateless API. User input is discarded immediately after response. No storage, no logging, no model training on user data.
Rate: 5 submissions per IP per day, 3000 characters per submission.
